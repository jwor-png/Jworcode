# Artefacts and Gives

> To be filled by John. This file lists the assets Pep can deploy as gives — the thing that earns the table.

## Structure to complete

For each artefact or give, provide:
- **Name** — what you call it
- **Format** — PDF, deck, interactive tool, live demo, diagnostic, framework
- **Who it is for** — which buyer profile, which sector
- **What it demonstrates** — the specific capability or value it lands
- **Licensing** — is this yours to use freely, or restricted?
- **Conversion intent** — what conversation does it open? What does it convert to?
- **Tier** — Cheap give (cold / mass) / Premium give (high-propensity / tailored)
